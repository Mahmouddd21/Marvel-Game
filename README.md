<h1>Marvel Game</h1>
<p>Marvel Game is a 2D 5x5 PVP game that features 15 champions divided into 3 types: hero, villain, and anti-hero. Each champion has 3 abilities.
</p>
<h1>Gameplay</h1>
The player picks 3 champions from the 15 available to be on their team. Each team will have a leader with a leader ability that can only be used once. The player wins the game after their opponentsâ€™ champions are dead.
